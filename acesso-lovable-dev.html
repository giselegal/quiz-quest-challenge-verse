Acesso ao Lovable.dev

Para facilitar o acesso ao ambiente Lovable.dev do Quiz Sell Genius, arraste os seguintes bookmarklets para sua barra de favoritos:

Admin Lovable Editor Visual

Instruções:

  1. Certifique-se que sua barra de favoritos está visível (pressione Ctrl+Shift+B ou Cmd+Shift+B se não estiver)
  2. Arraste os botões coloridos acima para sua barra de favoritos
  3. Para usar, simplesmente clique no favorito quando quiser acessar o ambiente Lovable.dev

Você também pode acessar diretamente a URL: https://giselegalvao.com.br

Para mais informações sobre o uso do Lovable.dev, consulte o arquivo LOVABLE-ACCESS.md no repositório.
